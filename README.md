## Laravel/Vue3/Inertia-Demo


This is a playground for
- Laravel
- Vue3
- InertiaJS


## HowTo

####Add composer dependencies
```
composer update
```

####Add npm dependencies
```
npm install
```

create an .env-file and add a database

#### call migrations
```
php artisan migrate:fresh --seed
```

#### run dev
```
npm run dev
```
